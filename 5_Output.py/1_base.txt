THERE ARE TWO TYPES OF LLM, ONE WHICH GIVES STRUCTURED OUTPUT AND ONE THAT DOESNT
THERE ARE FUNCTIONS FOR THESE
THE LLM THAT GIVES STRCUTURED OUTPUT, WE CAN USE with_structured_output, TO GET OUTPUT IN DESIRED FORMAT
THE LLM THAT DOESNT GIVE STRUCTURED OUTPUT, WE CAN USE output_parser, TO GET OUTPUT IN DESIRED FORMAT
